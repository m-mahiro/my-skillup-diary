以下に **Linuxで使用される代表的なVimコマンド一覧** を体系的にまとめます。
Vim（Vi IMproved）はViエディタを拡張した強力なテキストエディタで、**モード**によって操作方法が異なります。
主に以下の3つのモードがあります：

* **ノーマルモード**（コマンド実行・移動）
* **挿入モード**（文字入力）
* **コマンドラインモード**（保存・終了など）

---

## 1. 起動・終了系コマンド

| 操作内容              | コマンド           | 説明             |
| ----------------- | -------------- | -------------- |
| ファイルを開く           | `vim filename` | 指定したファイルを開く    |
| ファイルを新規作成         | `vim newfile`  | 新しいファイルを作成して開く |
| 保存                | `:w`           | 上書き保存          |
| 名前を付けて保存          | `:w newname`   | 指定した名前で保存      |
| 終了                | `:q`           | 終了（未保存なら警告）    |
| 強制終了              | `:q!`          | 保存せずに強制終了      |
| 保存して終了            | `:wq` または `:x` | 保存して終了         |
| 全保存終了             | `:wa`          | すべてのバッファを保存    |
| Vimを開いたまま外部コマンド実行 | `:!ls`         | 例：`ls`を実行      |

---

## 2. モード切替

| 操作内容          | キー    | 説明          |
| ------------- | ----- | ----------- |
| 挿入モードに入る      | `i`   | カーソルの前に挿入   |
| 挿入モードに入る（末尾）  | `A`   | 行末で入力開始     |
| 挿入モードに入る（次の行） | `o`   | 次の行を開く      |
| ノーマルモードに戻る    | `Esc` | どのモードからでも戻る |
| ビジュアルモードに入る   | `v`   | 範囲選択開始      |
| 行全体選択         | `V`   | 行単位で選択      |

---

## 3. カーソル移動

| 操作内容    | キー     | 説明                |
| ------- | ------ | ----------------- |
| 左       | `h`    | 1文字左へ             |
| 下       | `j`    | 1行下へ              |
| 上       | `k`    | 1行上へ              |
| 右       | `l`    | 1文字右へ             |
| 行頭へ     | `0`    | 行の最初へ             |
| 行末へ     | `$`    | 行の最後へ             |
| 次の単語へ   | `w`    | 単語の先頭へ進む          |
| 前の単語へ   | `b`    | 単語の先頭へ戻る          |
| 画面上部    | `H`    | 現在の画面の最上部         |
| 画面中央    | `M`    | 現在の画面の中央          |
| 画面下部    | `L`    | 現在の画面の最下部         |
| 指定行へ    | `:行番号` | 例：`:15` → 15行目へ移動 |
| ファイル末尾へ | `G`    | 最終行へ              |
| ファイル先頭へ | `gg`   | 先頭行へ              |

---

## 4. 編集（挿入・削除・コピー・貼り付け）

| 操作内容                | コマンド                       | 説明               |
| ------------------- | -------------------------- | ---------------- |
| 1文字削除               | `x`                        | カーソル上の文字削除       |
| 行削除                 | `dd`                       | 現在行を削除           |
| 3行削除                | `3dd`                      | 下3行を削除           |
| コピー                 | `yy`                       | 現在行をコピー（yank）    |
| 3行コピー               | `3yy`                      | 下3行をコピー          |
| 貼り付け（次行）            | `p`                        | カーソルの次に貼り付け      |
| 貼り付け（前行）            | `P`                        | カーソルの前に貼り付け      |
| 元に戻す                | `u`                        | Undo（直前の変更を取り消す） |
| やり直し                | `Ctrl + r`                 | Redo（Undoの取り消し）  |
| 1行挿入                | `o`                        | 次の行を開いて挿入        |
| 複数行コメントアウト（例：C言語など） | `Ctrl + v` → `I//` → `Esc` | 矩形選択を使う          |

---

## 5. 検索・置換

| 操作内容    | コマンド             | 説明             |
| ------- | ---------------- | -------------- |
| 前方検索    | `/word`          | “word” を前方検索   |
| 後方検索    | `?word`          | “word” を後方検索   |
| 次の一致へ移動 | `n`              | 同方向に次を検索       |
| 前の一致へ移動 | `N`              | 逆方向に次を検索       |
| 置換（1行）  | `:s/foo/bar/`    | “foo”を“bar”に置換 |
| 置換（全行）  | `:%s/foo/bar/g`  | 全文で置換          |
| 置換時に確認  | `:%s/foo/bar/gc` | 1つずつ確認しながら置換   |

---

## 6. ウィンドウ・バッファ操作

| 操作内容        | コマンド                 | 説明            |
| ----------- | -------------------- | ------------- |
| 新しいウィンドウを分割 | `:split` または `:sp`   | 横方向に分割        |
| 縦方向に分割      | `:vsplit` または `:vsp` | 縦方向に分割        |
| ウィンドウ間移動    | `Ctrl + w + w`       | 次のウィンドウに移動    |
| ウィンドウを閉じる   | `:q`                 | 現在のウィンドウを閉じる  |
| バッファ一覧      | `:ls`                | 開いているファイル一覧表示 |
| バッファ切替      | `:b n`               | n番目のバッファに切替   |
| 全バッファ閉じる    | `:qa`                | 全部閉じて終了       |

---

## 7. ファイル操作・外部連携

| 操作内容      | コマンド          | 説明           |
| --------- | ------------- | ------------ |
| 外部コマンド実行  | `:!コマンド`      | 例：`:!ls`     |
| ファイル再読み込み | `:e!`         | 変更を破棄して再読み込み |
| ファイル読み込み  | `:r filename` | ファイルの内容を挿入   |

---

## 8. マクロ・繰り返し

| 操作内容         | コマンド     | 説明                |
| ------------ | -------- | ----------------- |
| マクロ記録開始      | `q[a-z]` | 例：`qa`でレジスタaに記録開始 |
| マクロ記録終了      | `q`      | 記録終了              |
| マクロ実行        | `@a`     | レジスタaを実行          |
| 直前マクロ再実行     | `@@`     | 直前のマクロを再実行        |
| 直前のコマンドを繰り返す | `.`      | 直前の編集操作を再実行       |

---

## 9. 表示・ナビゲーション

| 操作内容          | コマンド                        | 説明          |
| ------------- | --------------------------- | ----------- |
| 行番号表示         | `:set number`               | 行番号を表示      |
| 行番号非表示        | `:set nonumber`             | 行番号を非表示     |
| シンタックスハイライト有効 | `:syntax on`                | 構文強調を有効化    |
| 折り返し設定        | `:set wrap` / `:set nowrap` | 改行表示の切替     |
| タブ幅変更         | `:set tabstop=4`            | タブを4スペースに設定 |
| インデント幅設定      | `:set shiftwidth=4`         | 自動インデント幅設定  |

---

## 10. Vimの終了ショートカットまとめ

| 操作内容   | コマンド           |
| ------ | -------------- |
| 保存して終了 | `:wq` または `ZZ` |
| 保存せず終了 | `:q!` または `ZQ` |
| 保存のみ   | `:w`           |
